# Icons Handoff — Unicode и эмодзи → SVG

Документ для дизайнера. В нём собраны **все места** в приложении, где сейчас используются юникод-символы (♈ ☉ ✦ и т.д.) и эмодзи (🌙 🔮 ✨…), которые нужно заменить на кастомные SVG-иконки.

Для каждого пункта указано: что это, где используется (контекст в UI), и желаемый размер.

---

## Технические требования к SVG

Все иконки должны быть в едином стиле, чтобы вписываться в celestial-эстетику приложения.

| Параметр | Значение |
|---|---|
| ViewBox | `0 0 24 24` (стандарт), для крупных глифов `0 0 32 32` |
| Стиль | Outline (только обводка, без заливки) |
| Stroke | `currentColor` (чтобы цвет наследовался от CSS) |
| stroke-width | `1.6` |
| stroke-linecap | `round` |
| stroke-linejoin | `round` |
| Палитра (для preview) | Золото `#E8C862` / приглушённое `#C4A35A` / тусклое `#8B6B3A` |

**Уже сделано:** 12 знаков зодиака — см. `frontend/public/zodiac-glyphs/` (aries.svg, taurus.svg, …) и `frontend/public/zodiac/` (для loader). Их можно использовать как референс стиля.
