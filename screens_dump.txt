================================================================
ASTRO TMA — Описание всех экранов приложения
Дата: 2026-04-05
================================================================

Всего: 12 экранов
Frontend: React 18 + TypeScript + Vite + Framer Motion
Backend: FastAPI + PostgreSQL + Redis
Деплой: Vercel (frontend) + VPS Docker (backend)

================================================================
1. ГЛАВНАЯ (Home)
================================================================
Файл: frontend/src/components/screens/Home.tsx
Навигация: tab "Сегодня" в нижней панели
API: horoscopeApi, tarotApi, horoscopeApi.getMoon()

Основной дашборд. Приветствие + дата, гороскоп с табами
(Сегодня/Завтра/Неделя/Месяц), карта таро дня с flip-анимацией,
фаза луны, энергетические бары.
Премиум: Завтра (25 Stars), Неделя (50), Месяц (75).

================================================================
2. ПРАКТИКИ (Discover)
================================================================
Файл: frontend/src/components/screens/Discover.tsx
Навигация: tab "Практики"

Хаб всех функций в 4 секциях:
- Карты: Таро, МАК-карты
- Отношения: Совместимость, Синастрия (locked)
- Луна: Лунный календарь, Транзиты (locked)
- Знания: Астро-новости (locked), Глоссарий (locked)

================================================================
3. НАТАЛЬНАЯ КАРТА (Natal)
================================================================
Файл: frontend/src/components/screens/Natal.tsx
Навигация: tab "Моя карта"
API: natalApi.getSummary(), natalApi.getFull()

Бесплатно: Солнце, Луна, Асцендент.
Премиум (150 Stars): 7 планет + дома + аспекты + LLM-интерпретация.
Требует данные рождения.

================================================================
4. ЛУННЫЙ КАЛЕНДАРЬ (Moon)
================================================================
Файл: frontend/src/components/screens/Moon.tsx
Навигация: Discover -> Лунный календарь
API: horoscopeApi.getMoon(), getMoonCalendar(year, month)

Фаза луны + процент, месячная сетка с emoji,
выбор дня с интерпретацией.

================================================================
5. ТАРО — Выбор расклада (Tarot)
================================================================
Файл: frontend/src/components/screens/Tarot.tsx
Навигация: Discover -> Таро
API: tarotApi.draw(spreadType)

4 расклада:
- Прошлое-Настоящее-Будущее: 3 карты, бесплатно
- Кельтский Крест: 10 карт, 30 Stars
- Неделя: 7 карт, 40 Stars
- Отношения: 5 карт, 30 Stars

Для Кельтского Креста: информационная страница с описанием
10 позиций (Крест + Посох) перед раскладом.

================================================================
6. РАСКЛАД 3 КАРТЫ (ThreeCardFlow)
================================================================
Файл: frontend/src/components/screens/ThreeCardFlow.tsx
Навигация: Tarot -> 3 карты

Анимированное колесо 22 карт. Фазы:
1. SPINNING — вращающееся кольцо, карты вразнобой
2. FLY-IN — карта летит + 3D flip + overlay + glow
3. REVEALED — имя + ключевые слова + кнопка
4. FLY-OUT — карта летит в слот лицом вверх
5. READING — все 3 карты + значения

================================================================
7. РАСКЛАДКИ (SpreadLayout)
================================================================
Файл: frontend/src/components/screens/SpreadLayout.tsx
Навигация: Tarot -> Celtic Cross / Relationship / Week

Кельтский Крест:
           [5/Цели]           [10/Итог]
  [4/Прошл] [1+2]  [6/Будущ]  [9/Надежды]
           [3/Подсозн]        [8/Окруж]
                               [7/Отнош]

Отношения:
     [1/Вы]      [2/Партнёр]
         [3/Связь]
         [4/Вызов]
         [5/Потенциал]

Неделя:
  [Пн] [Вт] [Ср] [Чт]
     [Пт] [Сб] [Вс]

Карты 68x102px, flip при тапе, номер на рубашке.

================================================================
8. СОВМЕСТИМОСТЬ (Compatibility)
================================================================
Файл: frontend/src/components/screens/Compatibility.tsx
Навигация: Discover -> Совместимость
API: compatibilityApi.get(signA, signB)

Два знака, общий %, 4 категории (Любовь, Общение,
Доверие, Страсть), сильные стороны и вызовы.

================================================================
9. ЗЕРКАЛО ДУШИ — МАК-карты (Mac)
================================================================
Файл: frontend/src/components/screens/Mac.tsx
Навигация: Discover -> Зеркало Души
API: macApi.draw()

Медитативное вытягивание. Emoji + название + описание +
вопрос для размышления + аффирмация. 3D flip.

================================================================
10. ПРОФИЛЬ (Profile)
================================================================
Файл: frontend/src/components/screens/Profile.tsx
Навигация: tab "Профиль"
API: usersApi.setupBirth(), setGender(), upsertMe()

Аватар, имя, пол, дата/время рождения, город с автокомплитом.
Редактирование триггерит пересчёт натальной карты.

================================================================
11. ОНБОРДИНГ (Onboarding)
================================================================
Файл: frontend/src/components/screens/Onboarding.tsx
API: usersApi.upsertMe(), setGender(), setupBirth()

5 шагов: Приветствие -> Пол -> Дата рождения -> Город -> Знак.
Прогресс-точки, анимированные переходы.

================================================================
12. SPLASH SCREEN
================================================================
Встроен в App.tsx

Заставка: фон + "ASTRO" + подзаголовок + 3 точки загрузки.
Fade-out после инициализации.
